California NPS Planner

This site allows for users to find the closest and furthest National Parks within the state of California based on their location. User can then get descriptive information on the park by clicking on the park's card. User can favorite the parks they like most which will be saved to their inputted email. 

Technologies used include: HTML, CSS, Javascript, Web APIs, Bulma, and more!
